## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies: 

- Next 13 (App Router)
- React
- Tailwind (Shadcn UI)
- Drizzle
- PostgreSQL
- Open AI (Dall-E, Chat GPT)
- Clerk
- Firebase
- Neon (db)
- Tip Tap
- TypeWriter Effect
